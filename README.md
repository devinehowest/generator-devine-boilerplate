# generator-devine-project

> [Yeoman](http://yeoman.io) generator

## Getting Started

### Install Yo

```bash
yarn global add yo
```

### Install generator-devine-project

```bash
yarn global add generator-devine-project
```

## Usage

Initiate the generator:

```bash
yo devine-project
```

### NPM scripts


```bash
npm run development
```

```bash
npm run production
```

```bash
npm run serve
```

```bash
npm run clean
```
